# Supabase Email Template for Password Reset (No OTP)

## Template Configuration

### In Supabase Dashboard:
1. Go to Authentication -> Email Templates
2. Select "Reset password" template
3. Update with this content:

### Subject:
Reset Your Password - Yang Chow Restaurant

### HTML Body:
```html
<h2>Reset Your Password</h2>
<p>Hello,</p>
<p>You have requested to reset your password for your account at Yang Chow Restaurant.</p>
<p>Please click the link below to reset your password:</p>
<p><a href="{{ .ConfirmationURL }}" style="background-color: #E81E0D; color: white; padding: 12px 24px; text-decoration: none; border-radius: 4px; display: inline-block;">Reset Password</a></p>
<p>Or copy and paste this link in your browser:</p>
<p>{{ .ConfirmationURL }}</p>
<p>If you did not request a password reset, please ignore this email.</p>
<p>Thank you!<br>Yang Chow Restaurant Team</p>
<hr>
<p style="color: #666; font-size: 12px;">This is an automated message. Please do not reply to this email.</p>
```

## Supabase Settings

### Authentication -> Settings:
- **Site URL**: `https://your-domain.com`
- **Redirect URLs**: 
  - `https://your-domain.com/reset-password`
  - `io.supabase.flutter://reset-callback/`

### Email Provider:
- **Custom SMTP** (SendGrid)
- **SMTP Host**: `smtp.sendgrid.net`
- **SMTP Port**: `587`
- **SMTP User**: `apikey`
- **SMTP Password**: `[SendGrid API Key]`

## How It Works:

1. User clicks "Forgot Password"
2. Enter email
3. Supabase sends reset email with direct link
4. User clicks link in email
5. App opens to `/reset-password` page
6. User sets new password directly (no OTP)
7. Password updated successfully

## Features:
- No 8-digit code required
- Direct password reset from email
- Secure and user-friendly
- Works with SendGrid
